# ConversorDec2Bin
# Código para convertir de decimal a binario

### Primero te pedirá el número decimal que quieres convertir a binario

### Luego te pedirá el número de "números" que quieres que muestre el código binario

### Si el número binario necesita más números a mostrar, añadirá los que necesite

### En un principio el programa no tiene problemas si pones un número en negativo

### Al final el programa te dará el número que le diste transformado en binario
